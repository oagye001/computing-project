How to use my application.

When run the main user interface window will appear on the screen. 
If not there should be a leaf icon below that click on to bring up the window. It has two buttons, county emissions and predicted emissions.

If the country emissions button is pressed it will switch to country emissions page. 
How to use the country emissions page:
There is a search bar for you to enter a country that exists within the EDGAR dataset.
Once you have entered your country click on the search button and it will display the emissions graph.
If you want to return to the main/home page click on the start page button

If the predicted emissions button is pressed it will switch to country emissions page. 
How to use the predicted emissions page:
There is a search bar for you to enter a number which represents the number of months you want to predict for
Once you have your number click on the submit button and it will display the graph of predicted values.
If you want to return to the main/home page click on the start page button
